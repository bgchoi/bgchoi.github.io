github 사용법

02 June 2016

리눅스 환경에서 git 사용법에 대해서 간략히 설명한다. 추후 로컬 환경 구축시 유용한 자료가 될것이다.

서버에 이미 git 레포지토리가 존재 하고 그 환경을 로컬에서 구성할 때

아래의 명령어를 순차적으로 수행:

$ git init
$ git add * or git add index.html
$ git commit -m 'comment here'
$ git remote add origin <원격 서버 주소>
$ git push origin master --force or git push origin master -f

  • git init .git 디랙토리를 만들어 git 환경을 구성한다.

  • git add head에 파일을 추가한다. 로컬에만 반영됨.

  • git commit head에 파일을 추가한다. 실제로 반영됨.

  • git remote add orgin git 환경파일에 원셕 서버 정보를 등록한다. git remote -v 명령어로 추가된 정보를 볼 수 있다.

  • git push origin master 원격 서버에 업로드 한다.
